# PlotFinder

PlotFinder is used to extract coordinate data from plots.

Plot types: 
<ul>
  <li>linear</li>
  <li>logarithmic</li>
  <li>mixed plots (lin-log / log-lin)</li>
</ul>

Also works for "tilted" plots, so perfect for graphs scanned from reference manuals.
